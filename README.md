# FORMATDISC MVP Simulation Tool

AI-powered MVP simulator for entrepreneurs. Validate your startup idea before you build.

**Live:** [www.formatdisc.hr](https://www.formatdisc.hr)

## Features

- **Financial Projections** - 12-month revenue, costs, and profit forecasts
- **Risk Analysis** - Comprehensive confidence scoring with factor breakdown
- **Growth Modeling** - Customer acquisition, MRR projections, break-even analysis
- **GO/NO-GO Verdict** - Clear, data-driven recommendations
- **PDF Export** - Professional investor-ready reports
- **Instant Results** - Complete analysis in seconds

## Quick Start

\`\`\`bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
\`\`\`

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Pages

- `/landing` - Marketing landing page
- `/` - Main simulator application

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Connect to Vercel
3. Set custom domain: `www.formatdisc.hr`
4. Deploy

### Environment Variables

No environment variables required for basic functionality.

## Technology Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **UI:** Radix UI + shadcn/ui
- **Charts:** Recharts
- **PDF:** jsPDF
- **Analytics:** Vercel Analytics
